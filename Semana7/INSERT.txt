INSERT: Se utiliza para agregar nuevos registros a una tabla. Por ejemplo, 
INSERT INTO Empleados (Nombre, Edad, Puesto) VALUES ('Juan Pérez', 30, 'Desarrollador');.

DELETE: Sirve para eliminar datos de una tabla. Es fundamental usar la cláusula 
WHERE para evitar la eliminación accidental de todos los registros de la tabla. Un ejemplo es: 
DELETE FROM Empleados WHERE Nombre = 'Juan Pérez';.

UPDATE: Permite actualizar o modificar datos existentes en una tabla. Al igual que con DELETE, es crucial usar la cláusula 
WHERE para especificar qué registros se deben actualizar. Por ejemplo: 
UPDATE Empleados SET Edad = 31 WHERE Nombre = 'Juan Pérez';.

PIVOT: Esta función se utiliza en SQL Server para convertir datos de filas a columnas. Esto es útil para transformar datos y mejorar la interpretación en reportes y análisis. El resultado es una tabla que muestra valores agregados (como una suma) para cada combinación, con las filas originales convertidas en columnas nuevas
